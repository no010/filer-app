# 你的「下载」文件夹，是不是早就没救了？

*200 个文件挤成一团，找个 datasheet 翻五分钟，想清理又不知道哪个还有用——每个人的下载目录，都是电脑里最快失控的角落。*

直到我写了个小工具，替我盯着、替我归类、替我提醒该清的。它叫 **filer**。

## 你大概也这样

打开「下载」文件夹，扑面而来的是：

- `STM32F103C8_Datasheet.pdf`、`STM32Fatasheet.pdf`、`datasheet(1).pdf`——其实是同一份下了三遍；
- 上个月那张发票，和半年前那张，名字长得几乎一样；
- 一个 4 个 G 的 Ubuntu 镜像、一个 27 个 G 的模型文件，占着地方从没再打开过；
- 还有几十个 `setup(1).exe`、`新建文件夹`、`未命名.png`……

你想找东西，找不到；你想清理，看着满屏不知道从哪下手。于是关掉，明天再说。明天复明天，下载目录就这么一直爆着。

**问题不是你不爱整理，是整理太费劲。** 手动改名、建文件夹、拖进去——一次两次行，每周都来一遍，没人受得了。

filer 要做的，就是把这件费劲事自动化：**你只管下，整理交给它。**

## filer 是什么

一句话：**监控你的下载目录，文件下完自动分析、建议归档位置，你确认后按规则整理进结构化目录树。**

跑起来是这样：

1. **后台盯着下载目录**。文件下完（大小稳定后）自动入箱，`.part`/`.crdownload` 这些临时文件直接忽略。
2. **自动认出它是啥**。PDF 读出页数、标题、甚至**厂商**（STM32→ST、ESP32→Espressif）；图片读尺寸和 EXIF（拍摄时间/相机/GPS）；office 文档读标题作者；压缩包列出里面有什么。几十 G 的大文件用稀疏指纹几秒判完，不卡。
3. **规则匹配**。按扩展名/关键词/厂商算出「分类 + 目标子目录 + 文件名」。比如 `STM32F103C8 Datasheet.pdf` → 分类 `Datasheet`，建议进 `D:\Filer\Datasheets\ST\`。规则能自己改。
4. **收件箱确认**。所有文件先进收件箱，你逐条过——确认/改/跳过，绝不误移。还能键盘三连：`j/k` 上下、`f` 归档、`i` 跳过、`e` 编辑，熟了之后飞快。
5. **整理归档**。确认后移动（或复制）到结构化目录，下载目录重新干净。

## 几个真的省心的设计

**重复下载不用慌**。同一份文件下了两遍？filer 用内容指纹识别，弹 4 个选项：跳过 / 保留两者（自动加后缀）/ 替换旧的 / 直接删新下的。你再也不会在三个文件夹里各存一份同一个 datasheet。

**长期没用的，主动提醒你清**。「回顾」页按记忆曲线式分层：<30 天「近期」、90 天「久未打开」、180 天「建议清理」、365 天「很可能不需要」。点「打开」续期、点「仍需要」重置、或直接送回收站。不是粗暴自动删，是把该清的摆到你面前，你拍板。

**想搜就搜**。跨文件名/分类/标签/元信息检索——输「stm32」找所有 ST 的 datasheet，输「发票」找所有票据，输相机型号能从照片 EXIF 里命中。

**自动归档（可选）**。嫌逐条确认烦？开「自动归档」，下完直接弹确认小窗，预填好建议，补个标签回车就走。

**还能看账**。统计页给你一张归档习惯的体检表——多少已归档、近 7 天归了多少、哪些分类占大头、哪些文件陈旧该清。

## 为什么不用网盘 / 下载管理器 / 手动整理？

- **网盘**要上传、要账号、要看你网盘脸色，文件还不一定在你手上。
- **下载管理器**（IDM/aria2）管的是「怎么下得快」，不管下完之后归哪儿。
- **手动整理**——上文说了，没人坚持得下来。

filer 补的就是这块空白：**下完之后到归档之间那一段**。本地跑，不上云、不联网（除了查更新）、不要账号。

---

## 本地、开源、不上云

所有索引存本地 SQLite，文件在本地目录间移动，filer 自己不传任何东西到任何服务器。代码 MIT 开源在 GitHub，能自己编译、自己审、自己改。删文件走**回收站**不是永久删，手滑有救；撤销能把归错的移回去。

技术栈：Tauri 2（Rust）+ React + Vite + Tailwind，单二进制，装完即用，不装 Python、不装运行时、不后台常驻大进程。

## 怎么获取

- **源码 + 安装包**：GitHub 搜 `no010/filer-app`（https://github.com/no010/filer-app），Releases 页有三平台安装包：
  - Windows：`.exe`（NSIS）/ `.msi`
  - macOS：`.dmg`（Apple Silicon + Intel 通用）
  - Linux：`.AppImage` / `.deb` / `.rpm`
- 首次打开选「下载监听目录」+「归档根目录」，存好就进主界面。往下载目录丢个文件，或点「立即扫描」回填已有文件，立刻看到效果。
- 开了**自动更新**，以后出了新版会弹窗提示，点一下下载重启。

> macOS/Windows 包当前未签名，首次打开右键→打开 / SmartScreen 选「仍要运行」即可（签名在办：Apple Developer + 开源免费签名 SignPath）。
