# 你的「下载」文件夹，是不是又爆了？

> 一个本地、开源、不上云的小工具 filer，帮你把乱成一锅粥的下载内容自动整理进结构化目录。

## 一个场景

你刚下完一个 STM32 的 datasheet、一张发票 PDF、一个 VSCode 安装包、还有上个月就该删的某个旧固件。它们整整齐齐地躺在 `C:\Users\你\Downloads` 里——和另外 200 个名字差不多的文件挤在一起。你想找上个月那份 datasheet，翻了五分钟没找到；你想清理一下，看着满屏文件又不知道哪个还有用、哪个是重复下的。

「下载」文件夹，是每个人电脑里最快失控的地方。

我们做了一个小工具来解决这件事：**filer**。

## filer 做了什么

一句话：**监控你的下载目录，文件下完自动分析、建议归档位置，你确认后按规则整理进结构化目录树。**

完整流程是这样的：

1. **自动监听**。filer 在后台看着你的下载目录，文件下完（size 稳定后）自动入箱，临时文件（`.part`/`.crdownload`）会忽略。
2. **自动分析**。识别文件类型（PDF/图片/office/压缩包…），PDF 还能读出页数、标题、甚至**厂商**（STM32 → ST、ESP32 → Espressif），图片读尺寸和 EXIF（拍摄时间/相机/GPS），office 读标题和作者，压缩包列出顶层条目。
3. **规则匹配**。按扩展名/关键词/厂商匹配内置规则，算出「分类 + 目标子目录 + 规范化文件名」。比如 `STM32F103C8 Datasheet.pdf` → 分类 `Datasheet`，建议归到 `D:\Filer\Datasheets\ST\`。规则可以自己在设置里改。
4. **收件箱确认**。所有文件先进收件箱，你逐条确认/编辑/跳过，不会误移。支持键盘三连：`j/k` 上下、`f` 归档、`i` 跳过、`e` 编辑。
5. **整理归档**。确认后按规则移动（或复制）到结构化目录，源目录保持干净。

## 几个用起来真的省心的设计

**重复下载别担心**。同一个文件你下了两遍？filer 用 sha256 内容指纹识别，命中已归档记录时弹 4 个选项：跳过 / 保留两者（加后缀）/ 替换旧的 / 直接删新下的。决策写回数据库，有据可查。几十 G 的大文件（镜像、模型）用稀疏多点指纹，几秒判完不卡扫描。

**长期没用的文件，主动提醒你清**。「回顾」页按记忆曲线式分层提示：<30 天「近期」、90 天「久未打开」、180 天「建议清理」、365 天「很可能不需要」。点「打开」续期、点「仍需要」重置、或直接送回收站。不是粗暴地自动删，而是把该清的摆到你面前。

**想搜就搜**。跨文件名/分类/标签/元信息全文检索——输「stm32」找到所有 ST 的 datasheet，输「发票」找到所有 Receipt，输某个相机型号能从照片 EXIF 里命中。

**自动归档（可选）**。嫌逐条确认烦？开「自动归档」，下载完成直接弹确认小窗，预填好建议，你补个标签回车就走，半自动。

**本地、开源、不上云**。所有索引存本地 SQLite，文件在本地目录间移动，filer 自己不联网（除了检查更新），不要账号、不传任何东西。代码 MIT 开源在 GitHub，你完全可以自己编译、自己审。

## 还有这些贴心点

- **时区可配**，默认用系统时区，日期目录按你的时区生成。
- **用户标签**，每条记录挂自定义标签，方便检索和归类。
- **Windows 右键**「用 filer 归档」，任意文件直接投递入箱。
- **拖拽导入**，往窗口拖文件就进箱。
- **统计仪表盘**，分类分布、陈旧度分布一眼看清你的归档习惯。
- **一键多选归档/删除**，批量处理一堆待办。
- **撤销**，归档错了能移回去；删除走**回收站**不是永久删，手滑有救。
- **开机自启 + 后台运行**，配右键菜单，随时整理。

## 技术栈

Tauri 2 (Rust) + React 18 + Vite + TypeScript + Tailwind。单二进制，无运行时依赖，装完即用。SQLite 做索引、`notify` 做文件监听、`lopdf` 读 PDF、`kamadak-exif` 读 EXIF、`chrono-tz` 处理时区。不装 Python、不装运行时、不后台常驻大进程。

## 怎么获取

- **源码**：GitHub `no010/filer-app`（MIT，欢迎 star / issue / PR）。
- **安装包**：Releases 页有 Windows（`.exe`/`.msi`）、macOS（`.dmg`，Apple Silicon + Intel 通用）、Linux（`.AppImage`/`.deb`/`.rpm`）。
- 首次打开会让你选「下载监听目录」+「归档根目录」，存好就进主界面。往下载目录丢个文件试试，或点「立即扫描」回填已有文件。

> macOS 包目前未签名，首次打开右键 → 打开；Windows 未签名，SmartScreen 提示「仍要运行」即可。签名正在办（Apple Developer + SignPath Foundation 开源免费签名）。

给所有「下载文件夹常年失控」的人——尤其是工程师（datasheet/固件/工具链一堆）、设计师（素材/字体/参考图一堆）、还有每个被「下载」目录淹没的普通用户。

如果你也受够了在 200 个文件里翻找，或者每隔几个月手动清理一次下载目录，试试 filer。让它替你盯着、替你归类、替你提醒该清的——你只管下，整理交给它。
